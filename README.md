🚁 無人機術科模擬器 - 智慧H點與視覺強化版

(UAV Certification Flight Simulator - Smart H-Pad & Visual Enhancements)

這是一款基於 Web 技術開發的 3D 無人機飛行模擬器，專為台灣民航局無人機專業操作證（基本一）術科考試量身打造。透過高擬真的物理引擎、動態環境干擾，以及精確的考場還原，幫助飛手在實際飛行前熟悉空間感與操作手感，大幅降低實機練習成本。
✨ 核心特色 (Key Features)
🎯 考場 1:1 精確還原

    標準場地佈置：包含精準比例的內/外圓（半徑 4m / 8m）、測驗區方框。

    P1 ~ P7 標示錐：具備漸層光影的亮橙色三角錐，並附帶螢幕空間投影的 3D 標籤，不論視角如何切換皆清晰可見。

    智慧 H 停機坪 (Smart H-Pad)：內建距離偵測邏輯，當機體進入停機坪半徑 1 公尺範圍內時，H 點會動態點亮為高亮度螢光綠，協助飛手判斷降落精準度。

🎮 擬真物理與操控 (Physics & Controls)

    Gamepad API 支援：完美支援實體遙控器（需具備 USB/藍牙搖桿訊號輸出），包含類比搖桿防飄移死區 (Deadzone) 處理。

    重力與推力運算：模擬真實機體慣性，油門 (Throttle) 必須大於重力值才能起飛，並具備地面防穿透鎖定。

    法規限高系統：內建 40 公尺強制虛擬圍籬 (Geofence) 限高。

    動態環境風力：支援 0-9 級風力模擬，具備亂流（X/Y/Z 軸隨機向量）與陣風推移效果。

👁️ 視覺與聽覺強化 (Audio & Visuals)

    Three.js 渲染引擎：支援 PCFSoftShadowMap 柔和動態陰影與機底圓形投影。

    多視角運鏡系統：

        站位視角 (追蹤)：模擬考官與飛手的固定站立點視角，可調整觀看高度與焦距。

        第三人稱跟隨：機背後方固定距離跟隨。

        第一人稱 (FPV)：模擬機頭鏡頭畫面。

    Web Audio API 動態音效：內建低通濾波器與多重振盪器 (Oscillator)，馬達聲浪會根據「油門負載」、「姿態傾角」以及「攝影機距離」即時運算音高與音量。

📊 專業遙測介面 (Telemetry HUD)

    左側飛行狀態儀表：即時顯示高度、X/Z 座標、視角狀態與環境參數。

    右側動態 JSON 監控：即時輸出姿態角 (Throttle, Yaw, Pitch, Roll) 的浮點數值與控制器按鍵觸發狀態。

    右下角全域俯視雷達：2D Canvas 繪製的雷達圖，顯示機頭朝向與考場相對位置。

🛠️ 技術棧 (Tech Stack)

    前端架構：純 HTML5 + CSS3 + ES6 JavaScript

    3D 引擎：Three.js (v0.160.0 透過 ES Module 引入)

    硬體互動：HTML5 Gamepad API

    音效處理：Web Audio API (動態合成，無需外部音檔)

⌨️ 鍵盤與滑鼠操作指南 (Controls)

系統控制

    點擊畫面 或 任意控制器按鍵：解鎖並啟動馬達音效引擎。

    M：靜音開關。

    P：開啟/關閉地面動態陰影（提升低階設備效能）。

    0 ~ 9：設定環境風力強度（0 為無風，9 為強風）。

視角控制

    C：切換攝影機模式（站位 / 跟隨 / FPV）。

    ↑ / ↓：（站位模式下）調整操作員觀看高度。

    I / O：（站位模式下）拉近/推遠鏡頭焦距 (Zoom In/Out)。

遙控器對應 (Gamepad)
請連接支援 XInput/DirectInput 的控制器或真實無人機遙控器（需具備模擬器傳輸線）：

    左搖桿 (Axis 2, 3)：油門 (Throttle) 與 偏航 (Yaw)
    右搖桿 (Axis 1, 0)：俯仰 (Pitch) 與 滾轉 (Roll)
    
已知支援的控制器: DJI RC3, Radiomaster Pocket
(註：各廠牌遙控器的 Axis 對應可能略有不同，可觀察右上方 JSON 遙測數據進行確認與微調。)
⚠️ 免責聲明

本模擬器旨在輔助空間感知與操作邏輯練習。實機飛行之氣動力學、電池壓降及硬體延遲與模擬環境必有差異。實際操作無人機時，請務必遵守當地民航法規，並在安全合法的空域進行。
